# Chocolate Shipped Dashboard 🍫📊

A **Power BI analytics project** focused on analyzing chocolate shipment data.
This project demonstrates the complete **Power BI workflow**, including data cleaning, modeling, DAX calculations, and report development to create an interactive business intelligence dashboard.

---

## 📁 Project File

* **ChocolateShipped.pbix** – Main Power BI report containing data models, measures, and visualizations.

---

## 🚀 Project Overview

This project covers the **end-to-end Power BI development process**, from preparing raw data to building an interactive dashboard that provides insights into shipment performance and trends.

---

## 🧹 Data Preparation

Data preparation and transformation were performed using **Power Query**, including:

* Data cleaning and transformation
* Adding and modifying columns
* Preparing structured tables for analysis

---

## 🧩 Data Modeling

The dataset was structured using **best practices in Power BI data modeling**, including:

* Creating a **Star Schema**
* Connecting tables with relationships
* Understanding **evaluation context**
* Implementing **calendar tables** for time-based analysis

---

## 📐 DAX & Calculations

Advanced calculations were implemented using **DAX**, including:

* Introduction to DAX measures
* Remixing and optimizing measures
* Using **VAR variables** in DAX
* Creating **Year-over-Year (YoY) comparisons**

---

## 📊 Data Visualization

Multiple Power BI visuals were used to present insights effectively:

* Line charts
* Donut and pie charts
* Treemap visualizations
* KPI card visuals
* Tables with custom formatting
* Top N filtering using visual filters

---

## 🎨 Report Design & Interactivity

The report was designed to provide an **interactive and user-friendly experience** through:

* Custom report mockups
* Visual and report-level filters
* Interactive slicers
* Conditional formatting
* Customized tooltips
* Image URLs in tables
* Adding images to reports
* Report interaction customization

---

## 🧪 Testing & Optimization

Final report testing included:

* Verifying visual interactions
* Ensuring accurate calculations
* Applying best practices for dashboard usability

---

## 💡 Key Learning Outcomes

* End-to-end **Power BI dashboard development**
* Practical use of **Power Query, Data Modeling, and DAX**
* Designing **interactive and visually effective BI reports**
* Applying **data storytelling techniques**
